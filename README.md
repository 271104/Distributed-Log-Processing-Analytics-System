## Architecture

Log Generator
↓
Redis Streams
↓
Consumer Group
↓
Log Processor
↓
PostgreSQL
↓
FastAPI Analytics API
↓
React Dashboard

## Tech Stack

- Python
- FastAPI
- Redis Streams
- PostgreSQL
- React
- Docker

## Features

- Real-time log generation
- Redis Streams event queue
- Consumer Group processing
- PostgreSQL persistence
- Analytics APIs
- Event distribution charts
- Timeline analytics
- Auto-refresh dashboard
